## 긴급 수정 필요 사항 (Critical)

### C-1. 스키마 `@type`이 `Person`으로 잘못 설정됨 — 즉각 수정

- **현황:** `"@type": "Person"` 으로 설정 (학원이 사람으로 인식됨)
- **영향:** Google Knowledge Panel 생성 불가, 로컬 팩 노출 불가, 모든 구조화 데이터 신뢰도 붕괴
- **수정:** `"@type": ["EducationalOrganization", "LocalBusiness"]` 로 교체

### C-2. `url` 필드 유효하지 않은 URI

- **현황:** `"url": "baewoo.kr"` (프로토콜 없음)
- **영향:** 스키마 유효성 검사 실패, 엔티티 식별 오류
- **수정:** `"url": "https://www.baewoo.kr"`

### C-3. sitemap.xml 완전 부재 (404)

- **현황:** sitemap.xml 404 반환 (baewoo.co.kr은 2021년 구형 사이트맵이라도 존재했으나 이 사이트는 전혀 없음)
- **영향:** AI 크롤러가 사이트 콘텐츠를 탐색할 방법 없음. 신규 콘텐츠 발견 불가
- **수정:** Gnuboard 사이트맵 플러그인으로 즉시 생성

### C-4. 합격현황 페이지 데이터 완전 부재

- **현황:** "합격현황" 페이지가 내비게이션에 존재하나 본문에 "게시물이 없습니다"
- **영향:** "다수합격생 배출달성" 주장과 빈 데이터 페이지가 공존 — AI가 주장을 허위로 판단할 수 있음. 학원법상 허위·과대광고 리스크
- **수정:** 검증 가능한 연도별 합격 데이터로 즉시 채우거나 페이지 숨김 처리

### C-5. llms.txt 부재 및 robots.txt Sitemap 미선언

- **현황:** llms.txt 404, robots.txt에 Sitemap 지시어 없음
- **영향:** AI 엔진이 사이트 맥락 추론 불가
- **수정:** llms.txt 생성 + robots.txt에 `Sitemap:` 추가
