## 핵심 요약

배우앤배움 키즈센터(baewoo.net)는 서울 서초구에 위치한 아역배우 전문 교육·캐스팅·매니지먼트 기관입니다. 사이트는 Gnuboard 5 기반 PHP CMS로 구축되어 서버사이드 렌더링이 잘 되어있지만, AI 검색 최적화 측면에서 심각한 결함이 있습니다.

### 강점
- 서버사이드 렌더링(SSR) — AI 크롤러가 모든 콘텐츠에 접근 가능
- 개방적인 robots.txt — AI 크롤러 차단 없음
- 활발한 SNS 활동 (인스타그램 팔로워 9,372명, 네이버 블로그, 유튜브, 페이스북)
- 실제 캐스팅 실적이 경험 증명 역할
- 사업자등록번호 및 학원 인가번호 표시

### 심각한 약점
- 스키마 마크업이 `Person`(사람)으로 설정됨 — `Organization`(조직)이어야 함. 근본적인 엔터티 오류
- sitemap.xml 없음 (모든 경로에서 404)
- llms.txt 파일 없음
- canonical URL 없음 — 중복 콘텐츠 위험 높음
- 모든 페이지에 동일한 메타 설명 사용
- AI가 추출할 수 있는 FAQ 콘텐츠 없음
- 강사 프로필·자격 정보 없음 (E-E-A-T 가장 취약한 부분)
- Reddit/Wikipedia/LinkedIn 노출 전무

### URL 구조 평가

사이트가 PHP 쿼리 파라미터 URL을 전면 사용:
- `/web/bbs/content.php?co_id=company`
- `/web/bbs/board.php?bo_table=new_drama`

기술적으로는 작동하지만 SEO 친화적이지 않습니다. 현대적 CMS는 `/company`나 `/casting/drama`처럼 클린 URL을 사용합니다. canonical 태그 없이 이러한 파라미터 기반 URL은 중복 콘텐츠 위험을 크게 높입니다.
