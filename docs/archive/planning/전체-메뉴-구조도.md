## 0. 진입 흐름 개요

```
[게이트 페이지]
       │
       ├─ 아트센터 선택 ──→ [아트센터 서브홈]
       ├─ 입시센터 선택 ──→ [입시센터 서브홈]
       ├─ 하이틴센터 선택 ─→ [하이틴센터 서브홈]
       ├─ 키즈센터 선택 ──→ [키즈센터 서브홈]
       └─ 애비뉴센터 선택 ─→ [애비뉴센터 서브홈]
```

## 1. 게이트 페이지 (`/` or `/gate`)

```
┌────────────────────────────────────────────────────────┐
│                   배우앤배움 BNB                        │
│              — 브랜드 인트로 / 슬로건 —                  │
├────────────┬──────────┬──────────┬────────┬────────────┤
│  아트센터   │  입시센터  │ 하이틴센터 │ 키즈센터 │  애비뉴센터  │
│ (성인/데뷔) │ (대입/예고) │(중고생 특화)│(아동 특화)│(40세 이상)│
└────────────┴──────────┴──────────┴────────┴────────────┘
│  대상별 바로가기: 아동 | 청소년 | 입시생 | 성인 | 40세 이상  │
│  목적별 바로가기: 배우 데뷔 | 입시 준비 | 캐스팅 | 상담 신청  │
└────────────────────────────────────────────────────────┘
```

## 2. 공통 헤더 구조 (센터 진입 후)

```
┌──────────────────────────────────────────────────────────────────┐
│  [▼ 센터 선택]          [BNB 로고]          [오디션] [상담신청]   │  ← Quick CTA
├──────────────────────────────────────────────────────────────────┤
│  배우앤배움  │  교육  │  캐스팅  │  아티스트  │  지원센터  │  ← GNB 1depth
└──────────────────────────────────────────────────────────────────┘
```

> **센터 선택 드롭다운**: 아트센터 / 입시센터 / 하이틴센터 / 키즈센터 / 애비뉴센터
> **CTA 버튼**: [오디션 지원하기] [무료 상담신청] — 항상 우측 상단 고정
> **입시센터**: 교육 / 합격현황 / 합격자소개는 입시센터 전용 구조를 사용한다.
> **정렬 기준**: 아트센터 메뉴를 기준으로 하이틴센터·키즈센터·애비뉴센터의 1depth를 맞춘다.

## 3. GNB 전체 메뉴 (1depth → 2depth → 3depth)

---

### 3-1. 배우앤배움 (소개)

| 2depth | 3depth | 비고 |
|--------|--------|------|
| 회사소개 | — | 브랜드 스토리, 운영 철학 |
| 대표인사말 | — | 하이틴·키즈 공통 |
| 센터소개 | — | 센터별 소개 페이지 |
| 시설안내 | — | 갤러리형 |
| 오시는 길 | — | 센터별 지도 |

```
배우앤배움
├─ 회사소개
├─ 센터소개
├─ 대표인사말              ← 하이틴·키즈
├─ 시설안내
└─ 오시는 길
```

### 3-2. 교육

```
교육
├─ 등급제 교육관리시스템       ← 아트·하이틴·키즈 공통
├─ 엔터테인먼트 위탁교육       ← 아트·하이틴·키즈 공통
├─ 교육진 소개
├─ 커리큘럼                    ← 아트·하이틴·애비뉴
├─ 하이틴센터 특강             ← 하이틴 전용
│
├─ [키즈센터 전용]
│   ├─ 영재 교육과정
│   ├─ 아역배우 교육과정
│   └─ 아티스트 교육과정
│
└─ [입시센터 전용]
    ├─ 입시 매니지먼트
    ├─ 특별한 시스템
    ├─ 교육진 소개
    └─ 커리큘럼
```

### 3-3. 캐스팅

```
캐스팅
├─ 드라마·광고 출연장면           ← 아트·하이틴·키즈 공통
├─ 캐스팅 출연현황        ← 아트·하이틴·키즈 공통
├─ 드라마 광고 캐스팅             ← 아트·하이틴·키즈 공통
├─ 캐스팅 시스템                  ← 아트
├─ 촬영·오디션 스케줄       ← 아트
├─ BNB캐스팅                      ← 키즈·하이틴
├─ IMGround                       ← 키즈·하이틴
├─ 캐스팅/모집 안내               ← 애비뉴
├─ 프로필 촬영                    ← 애비뉴
├─ BX모델에이전시                 ← 하이틴 전용
├─ 다이렉트 캐스팅                ← 하이틴 전용
├─ BNB 캐스팅 섭외뉴스             ← 하이틴 전용
│
└─ [입시센터 전용 - 합격현황]
    ├─ 대학교
    └─ 예술고등학교
```

├─ 아역배우 프로필                 ← 키즈센터 전용
├─ 배우 프로필                    ← 애비뉴
├─ 프로필 촬영·제작               ← 하이틴·키즈
│
└─ [입시센터 전용 - 합격자소개]
    ├─ 합격 후기
    └─ 합격 영상
```

### 3-5. 지원센터

```
지원센터
├─ NEWS & NOTICE
├─ 입학안내
├─ 학원100%이용법

## 4. 애비뉴센터 — 같은 1depth 안의 원페이지 구성

```
애비뉴센터 (baewoorun.co.kr → /avenue)
│
├─ 배우앤배움
│   ├─ 애비뉴센터 소개
│   ├─ 제휴업체
│   └─ 시설 안내
├─ 교육
│   ├─ 강사진 소개
│   └─ 커리큘럼
├─ 캐스팅
│   ├─ 캐스팅/모집 안내
│   └─ 프로필 촬영
├─ 아티스트
│   └─ 배우 프로필
└─ 지원센터
    ├─ NEWS & NOTICE
    ├─ 입학안내
    ├─ 학원100%이용법
    ├─ 스타카드 멤버쉽서비스
    └─ 자주하는 질문
```

> 애비뉴센터는 원페이지 앵커를 사용하되, GNB 1depth 명칭은 아트·하이틴·키즈와 동일하게 둔다.
