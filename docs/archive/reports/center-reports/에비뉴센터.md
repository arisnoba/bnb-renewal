## 긴급 수정 필요 사항 (Critical)

### C-1. SSL 인증서 오류 — AI 크롤러 완전 차단

- **현황:** `https://www.baewoorun.co.kr` 접근 시 SSL 인증서 검증 실패 (`unable to verify the first certificate`)
- **영향:**
  - GPTBot, ClaudeBot, PerplexityBot, Googlebot 등 **모든 AI/검색 크롤러가 HTTPS로 접근 불가**
  - AI 학습 데이터에 이 사이트 콘텐츠가 포함될 수 없음
  - 브라우저에서 보안 경고 노출 → 신뢰도 직접 훼손
  - HTTP 접근만 가능한 상태 (2026년 기준 비표준)
- **수정:** 유효한 SSL 인증서 발급·설치 (Let's Encrypt 무료 인증서 또는 호스팅사 SSL 서비스)
- **난이도:** 낮음 (호스팅사 관리 패널 → SSL 인증서 갱신·설치, 1~2시간)

### C-2. Schema.org 마크업 완전 부재

- **현황:** JSON-LD, Microdata, RDFa 어떤 형태의 구조화 데이터도 없음 (다른 BNB 계열사는 최소한 Organization 스키마라도 있음)
- **영향:** Google Knowledge Graph가 이 사이트를 엔티티로 인식 불가. AI 시스템의 브랜드·사업 정보 추출 불가
- **수정:** EducationalOrganization + LocalBusiness 스키마 즉시 적용 (아래 코드 참조)

### C-3. meta description 공백, OG 태그 모두 비어 있음

- **현황:**
  ```html
  <meta name="description" content="" /> ← 공백 <meta property="og:title" content="" /> ← 공백 <meta property="og:description" content="" /> ← 공백 <meta property="og:url" content="" /> ← 공백
  ```
- **영향:** 검색 결과 및 SNS 공유 시 제목·설명 없음. AI 요약 생성 품질 최저. `og:image`만 설정(`/img/baewoorun2.jpg`)되어 있어 의도는 있으나 나머지 필드 누락
- **수정:** 모든 메타 필드 채우기 (아래 참조)

### C-4. 단일 페이지 + JS 전용 내비게이션 — 크롤 가능한 URL 2개뿐

- **현황:** 전체 사이트가 사실상 단일 HTML 페이지. 내부 링크는 파비콘 2개(`/img/favicon.jpg`, `/img/16x16.ico`)뿐. 모든 섹션이 JavaScript로 처리
- **영향:** Googlebot·AI 크롤러가 다른 페이지로 이동할 수 없어 링크 그래프 없음. sitemap도 없어 콘텐츠 발견 경로 제로
- **수정 방안:**
  1. 주요 섹션을 별도 URL 페이지로 분리 (중장기)
  2. 또는 SSR(서버사이드 렌더링) 적용으로 콘텐츠를 크롤 가능하게 전환
  3. 단기: 모든 핵심 콘텐츠를 한 페이지에라도 HTML로 직접 마크업

### C-5. sitemap.xml 및 robots.txt Sitemap 지시어 없음

- **현황:** sitemap.xml 응답 없음 (SSL 오류로 확인 불가). robots.txt에 Sitemap 미선언
- **수정:** SSL 수정 후 sitemap.xml 생성, robots.txt에 `Sitemap:` 추가
