## 0. 진입 흐름 개요

```
[게이트 페이지]
       │
       ├─ 아트센터 선택 ──→ [아트센터 서브홈]
       ├─ 입시센터 선택 ──→ [입시센터 서브홈]
       ├─ 하이틴센터 선택 ─→ [하이틴센터 서브홈]
       ├─ 키즈센터 선택 ──→ [키즈센터 서브홈]
       └─ 애비뉴센터 선택 ─→ [애비뉴센터 서브홈]
```

## 1. 게이트 페이지 (`/` or `/gate`)

```
┌────────────────────────────────────────────────────────┐
│                   배우앤배움 BNB                        │
│              — 브랜드 인트로 / 슬로건 —                  │
├────────────┬──────────┬──────────┬────────┬────────────┤
│  아트센터   │  입시센터  │ 하이틴센터 │ 키즈센터 │  애비뉴센터  │
│ (성인/데뷔) │ (대입/예고) │(중고생 특화)│(아동 특화)│(40세 이상)│
└────────────┴──────────┴──────────┴────────┴────────────┘
│  대상별 바로가기: 아동 | 청소년 | 입시생 | 성인 | 40세 이상  │
│  목적별 바로가기: 배우 데뷔 | 입시 준비 | 캐스팅 | 상담 신청  │
└────────────────────────────────────────────────────────┘
```

## 2. 공통 헤더 구조 (센터 진입 후)

```
┌──────────────────────────────────────────────────────────────────┐
│  [▼ 센터 선택]          [BNB 로고]          [오디션] [상담신청]   │  ← Quick CTA
├──────────────────────────────────────────────────────────────────┤
│  배우앤배움  │  교육  │  캐스팅  │  매니지먼트  │  운영  │  상담센터  │  ← GNB 1depth
└──────────────────────────────────────────────────────────────────┘
```

> **센터 선택 드롭다운**: 아트센터 / 입시센터 / 하이틴센터 / 키즈센터 / 애비뉴센터
> **CTA 버튼**: [오디션 지원하기] [무료 상담신청] — 항상 우측 상단 고정

## 3. GNB 전체 메뉴 (1depth → 2depth → 3depth)

---

### 3-1. 배우앤배움 (소개)

| 2depth | 3depth | 비고 |
|--------|--------|------|
| 회사소개 | — | 브랜드 스토리, 운영 철학 |
| 대표인사말 | — | 입시·키즈·하이틴 공통 |
| 센터소개 | — | 아트센터 전용 (identity) |
| 연혁 | — | 입시센터 전용 |
| 자회사 안내 | — | 입시센터 전용 |
| 시설안내 | — | 갤러리형 |
| 오시는 길 | — | 센터별 지도 |

```
배우앤배움
├─ 회사소개
├─ 대표인사말              ← 입시·키즈·하이틴
├─ 센터소개                ← 아트센터 전용
├─ 연혁                    ← 입시센터 전용
├─ 자회사 안내             ← 입시센터 전용
├─ 시설안내
└─ 오시는 길
```

---

### 3-2. 교육

```
교육
├─ 등급제 교육관리시스템       ← 아트·키즈·하이틴 공통
├─ 교육진 소개 (강사 갤러리)
├─ 엔터테인먼트 위탁교육       ← 아트·키즈·하이틴 공통
├─ 이달의 커리큘럼             ← 아트·하이틴 공통 (Ajax 필터)
├─ 하이틴센터 특강             ← 하이틴 전용
│
├─ [입시센터 전용]
│   ├─ 입시반 커리큘럼
│   ├─ 재입시반 커리큘럼
│   ├─ 편입반 커리큘럼
│   ├─ 예비 입시반 커리큘럼
│   ├─ 예고 입시반 커리큘럼
│   └─ 입시 매니지먼트
│
└─ [키즈센터 전용]
    ├─ 영재 교육과정
    ├─ 아역배우 교육과정
    └─ 아티스트 교육과정
```

---

### 3-3. 캐스팅

```
캐스팅
├─ 이달의 드라마·광고 출연장면     ← 공통 (갤러리)
├─ 진행중인 캐스팅 출연현황        ← 공통
│
├─ [드라마 캐스팅]
│   ├─ 유캐스팅                   ← 공통
│   ├─ BNB캐스팅                  ← 키즈·하이틴
│   └─ IMGround                   ← 키즈·하이틴
│
├─ [광고 캐스팅]
│   ├─ BX모델에이전시              ← 공통
│   └─ 다이렉트 캐스팅             ← 하이틴 전용 (내부탭)
│
├─ BNB 캐스팅 섭외뉴스             ← 하이틴 전용
│
├─ [입시센터 전용 - 합격현황]
│   ├─ 대학교 합격현황
│   └─ 예술고등학교 합격현황
│
└─ 오디션 지원하기 (FORM)          ← 아트·하이틴
```

### 3-4. 매니지먼트

```
매니지먼트
├─ 매니지먼트 시스템 소개
├─ 매니저 소개                    ← 아트센터 전용
├─ 이달의 촬영·오디션 스케줄       ← 캘린더형

├─ BNB출신 아티스트               ← 아트센터 전용
├─ 아역배우 프로필                 ← 키즈센터 전용
├─ 촬영후기                       ← 아트센터 전용
│
└─ [입시센터 전용 - 합격자 소개]
    ├─ 수강생 합격후기
    └─ 수강생 합격영상
│
└─ 프로필 촬영·제작               ← 공통
```

## 4. 애비뉴센터 — 별도 구성 (원페이지)

```
애비뉴센터 (baewoorun.co.kr → /centers/avenue)
│
├─ Section 1: 센터 소개
├─ Section 2: 제휴업체 (스타카드 파트너)
├─ Section 3: 강사진 소개
└─ Section 4: 캐스팅·모집 안내
               └─ [상담신청] → 공통 상담 폼 연결
```

> 애비뉴센터는 현재 독립 원페이지 구조로, 리뉴얼 시 통합 사이트 편입을 위한 콘텐츠 보강 필요.
