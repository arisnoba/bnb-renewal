## 1. 관리자 접속과 계정

1. 웹 브라우저에서 관리자 주소로 접속합니다.
2. 전달받은 이메일과 비밀번호를 입력합니다.
3. 로그인 후 좌측 메뉴에서 관리할 항목을 선택합니다.

![로그인 화면 캡처](./images/01-login.png)

### 비밀번호를 잊은 경우

로그인 화면의 **비밀번호를 잊으셨나요?**를 선택해 안내에 따라 재설정합니다. 재설정 메일이 보이지 않으면 스팸함을 확인한 뒤 관리자에게 계정 이메일 주소를 전달합니다.

### 계정 사용 시 주의사항

- 계정은 담당자별로 사용하고, 비밀번호를 메신저나 문서에 기록하지 않습니다.
- 담당자가 변경되면 기존 계정의 권한을 먼저 점검합니다.
- 다른 센터 콘텐츠가 보이더라도, 본인 담당 센터와 무관한 항목은 수정하지 않습니다.

## 2. 관리자 화면 이해하기

관리자 화면은 아래 세 영역으로 구성됩니다.

| 영역 | 하는 일 |
| --- | --- |
| 좌측 메뉴 | 관리할 콘텐츠 종류를 선택합니다. |
| 목록 화면 | 등록된 항목을 검색하고, 수정하거나 새 항목을 만듭니다. |
| 편집 화면 | 제목, 본문, 이미지, 공개 상태를 입력하고 저장합니다. |

![관리자 전체 화면 캡처](./images/02-admin-overview.png)

목록 화면에서는 제목 또는 필터로 원하는 항목을 찾습니다. 새 항목은 **새로 만들기**로 등록하고, 기존 항목은 제목을 선택해 수정합니다.

## 3. 공통 작업 순서

콘텐츠 종류와 관계없이 아래 순서를 기본으로 사용합니다.

1. 좌측 메뉴에서 콘텐츠 종류를 선택합니다.
2. **새로 만들기**를 누르거나, 수정할 기존 항목을 선택합니다.
3. 제목과 필수 입력값을 먼저 작성합니다.
4. 본문과 이미지를 입력합니다.
5. 노출할 센터와 공개 상태를 확인합니다.
6. **저장**합니다.
7. 공개 사이트에서 실제 노출 결과를 확인합니다.

### 저장과 공개 상태

- **임시저장**: 작성 중인 내용입니다. 공개 사이트에 노출되지 않습니다.
- **공개**: 저장 후 공개 사이트에 노출됩니다.
- **비공개**: 기존 콘텐츠를 숨길 때 사용합니다. 삭제가 필요한지 확신이 없을 때는 먼저 비공개로 전환합니다.

> 콘텐츠마다 상태 이름이나 입력 위치가 다를 수 있습니다. 공개 여부가 보이지 않으면 해당 항목의 저장 후 공개 사이트 노출 여부를 먼저 확인하세요.

## 4. 이미지 업로드와 선택

이미지는 여러 콘텐츠에서 공통으로 사용합니다. 먼저 이미지를 업로드한 뒤, 콘텐츠 편집 화면에서 그 이미지를 선택합니다.

### 이미지 업로드

1. 이미지 선택 필드에서 **업로드** 또는 **새로 만들기**를 선택합니다.
2. 사용할 이미지 파일을 선택합니다.
3. 파일명과 미리보기를 확인한 뒤 저장합니다.
4. 원래 편집 화면으로 돌아와 업로드한 이미지를 선택합니다.

![이미지 업로드 화면 캡처](./images/04-media-upload.png)

### 이미지 작성 기준

- 가로형 이미지와 세로형 이미지를 역할에 맞게 사용합니다.
- 텍스트가 들어간 배너는 모바일에서 글자가 잘리지 않는지 확인합니다.
- 같은 이미지를 여러 번 올리기보다, 이미 등록된 이미지를 검색해 재사용합니다.
- 인물 사진과 상담 정보 등 개인정보가 포함된 이미지는 게시 전 사용 동의를 확인합니다.

## 5. 메인 배너 관리

**메인설정 → 메인 배너**에서 센터별 첫 화면 배너를 관리합니다.

### 배너 등록 또는 수정

1. **메인 배너**를 선택합니다.
2. 새 배너를 만들거나 수정할 배너를 선택합니다.
3. 배너 제목, 노출 센터, 이미지, 연결 주소를 입력합니다.
4. 노출 기간과 공개 상태가 있다면 함께 확인합니다.
5. 저장 후 해당 센터의 메인 화면에서 배너를 확인합니다.

![메인 배너 편집 화면 캡처](./images/05-main-banner.png)

### 확인할 사항

- 배너가 올바른 센터에만 노출되는지 확인합니다.
- 연결 주소를 입력했다면 클릭 후 이동 위치를 확인합니다.
- PC와 모바일에서 이미지가 잘리지 않는지 확인합니다.
- 동일한 캠페인 배너가 중복 등록되지 않았는지 목록에서 확인합니다.

## 6. 소식과 FAQ 관리

**운영/소식 → 소식**과 **운영/소식 → FAQ**에서 공지와 자주 묻는 질문을 관리합니다.

### 소식 등록

1. **소식**에서 새 항목을 만듭니다.
2. 제목, 요약, 본문을 입력합니다.
3. 필요하면 대표 이미지를 선택합니다.
4. 노출할 센터와 발행일, 공개 상태를 확인합니다.
5. 저장 후 공개 사이트 소식 목록과 상세 화면을 확인합니다.

### FAQ 등록

1. **FAQ**에서 새 항목을 만듭니다.
2. 질문과 답변을 입력합니다.
3. 노출할 센터와 분류를 확인합니다.
4. 저장 후 해당 센터 FAQ에서 질문이 열리고 답변이 읽히는지 확인합니다.

![소식 또는 FAQ 편집 화면 캡처](./images/06-news-faq.png)

## 7. 강사진과 아티스트 관리

**교육 → 강사진**, **아티스트 → 프로필**, **아티스트 → 아티스트 소식**에서 인물 콘텐츠를 관리합니다.

### 강사진 또는 프로필 등록

1. 새 항목을 만듭니다.
2. 이름과 소속 센터를 입력합니다.
3. 대표 이미지를 선택하고, 소개와 경력 등 필요한 정보를 입력합니다.
4. 공개 상태를 확인해 저장합니다.
5. 해당 센터의 강사진 또는 아티스트 페이지에서 확인합니다.

> 기존 인물 정보를 수정할 때는 같은 이름의 중복 항목을 새로 만들지 말고, 먼저 목록에서 기존 항목을 검색합니다.

## 8. 캐스팅/오디션 콘텐츠 관리

**캐스팅/오디션** 메뉴에서는 캐스팅, 출연작, 오디션 일정 등 활동 콘텐츠를 관리합니다.

### 기본 입력 순서

1. 목적에 맞는 메뉴를 선택합니다. 예: 캐스팅, 출연작, 오디션 일정
2. 제목과 작품 또는 일정 정보를 입력합니다.
3. 노출할 센터, 날짜, 대표 이미지, 본문을 입력합니다.
4. 연결할 인물 또는 방송사가 있다면 정확한 기존 항목을 선택합니다.
5. 공개 상태를 확인해 저장합니다.
6. 공개 사이트에서 센터, 날짜, 이미지, 본문이 올바르게 보이는지 확인합니다.

## 9. 입시센터 후기와 합격 콘텐츠 관리

**입시센터 후기/합격** 메뉴는 입시센터 전용입니다. 다른 센터 콘텐츠로 등록하지 않도록 주의합니다.

### 등록 전 확인

- 후기 또는 합격 콘텐츠 유형을 올바르게 선택했는지
- 학생 이름, 학교명, 전형 정보 등 공개 가능한 정보만 입력했는지
- 사진과 영상의 사용 동의를 확인했는지
- 공개 상태와 노출 위치를 확인했는지

등록 후에는 입시센터 공개 화면에서 제목, 이미지, 영상 또는 본문이 정상적으로 보이는지 확인합니다.

## 10. 상담 문의 확인

**상담 → 문의**에서 웹사이트로 접수된 상담 내용을 확인합니다.

### 문의 확인 순서

1. 목록에서 최근 접수 건 또는 담당 센터 필터를 선택합니다.
2. 문의 내용을 열어 신청자 정보와 희망 상담 내용을 확인합니다.
3. 내부 상담 절차에 따라 연락 및 처리 상태를 기록합니다.
4. 첨부 파일이 있으면 다운로드 전 파일명과 접수 내용을 확인합니다.

### 개인정보 주의사항

- 문의 내용과 첨부 파일은 상담 처리 목적 외로 공유하지 않습니다.
- 화면 캡처나 외부 문서에 신청자 연락처를 포함하지 않습니다.
- 처리 완료 후 보관 및 삭제는 회사의 개인정보 처리 기준을 따릅니다.

## 11. 발행 전 확인과 문제 해결

### 발행 전 체크리스트

- [ ] 제목과 본문에 오탈자가 없습니다.
- [ ] 올바른 센터에 노출되도록 선택했습니다.
- [ ] 대표 이미지와 본문 이미지가 의도한 항목인지 확인했습니다.
- [ ] 공개 상태와 발행일을 확인했습니다.
- [ ] 저장 후 공개 사이트에서 PC와 모바일 화면을 확인했습니다.

### 자주 발생하는 문제

| 상황 | 먼저 확인할 내용 |
| --- | --- |
| 저장했는데 사이트에 보이지 않음 | 공개 상태, 노출 센터, 발행일을 확인합니다. |
| 다른 센터에 노출됨 | 콘텐츠의 센터 선택값을 확인하고 수정합니다. |
| 이미지가 보이지 않음 | 이미지가 실제로 선택되어 저장됐는지, 파일 업로드가 완료됐는지 확인합니다. |
| 수정 전 내용이 보임 | 브라우저를 새로고침한 뒤 다시 확인합니다. 계속되면 관리자에게 콘텐츠 주소와 수정 시각을 전달합니다. |
| 삭제 여부가 망설여짐 | 바로 삭제하지 말고 먼저 비공개로 전환한 뒤 관리자에게 확인합니다. |

### 지원 요청 시 전달할 정보

문제가 해결되지 않으면 아래 정보를 함께 전달합니다.

- 문제가 발생한 관리자 메뉴와 콘텐츠 제목
- 공개 사이트 주소
- 어떤 작업 후 문제가 발생했는지
- 오류 화면 캡처
- 문제가 발생한 날짜와 시간
