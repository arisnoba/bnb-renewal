# BNB Industry 5개 계열사 웹사이트 SEO 리뉴얼 제안서

## 1. 제안 배경

BNB Industry는 아트센터, 입시센터, 키즈센터, 하이틴센터, 애비뉴센터까지 서로 다른 타깃과 검색 의도를 가진 5개의 브랜드를 운영하고 있습니다.

각 브랜드는 서비스 성격과 고객층이 분명히 다르기 때문에, 검색엔진과 AI 검색 환경에서도 각각의 브랜드가 독립적으로 잘 이해되고 노출될 수 있는 구조가 필요합니다.

이번 검토 결과, 현재 5개 사이트는 브랜드 자산과 콘텐츠 자산을 충분히 보유하고 있음에도 불구하고, 웹사이트 구조와 정보 설계 한계로 인해 검색 노출과 전환 성과를 충분히 만들어내지 못하는 상태로 판단되었습니다.

이에 따라 본 제안서는 `5개 사이트를 연결하는 단일 게이트 웹사이트 신설`보다, `각 브랜드 사이트의 리뉴얼`이 더 효과적인 방향이라는 점을 설명드리고자 합니다.

## 2. 현재 운영 현황 요약

### 브랜드별 진단 요약

| 브랜드 | 현재 상태 요약 |
| --- | --- |
| 아트센터 | 강사진, FAQ, 커리큘럼 등 자산은 충분하나 구조화와 기술 기반이 약해 검색 성과로 연결되지 못함 |
| 입시센터 | 합격/성과 자산이 핵심인데 구조화, 정보 신뢰, 페이지 설계 측면에서 보완이 필요함 |
| 키즈센터 | 브랜드 활동성은 있으나 메타, canonical, 스키마 등 검색 기본 체계가 미흡함 |
| 하이틴센터 | 콘텐츠 구조와 기술 기반이 가장 약한 편으로, 검색엔진 친화 구조 재정비가 필요함 |
| 애비뉴센터 | 단일 페이지 구조와 기술 요소 미비로 인해 검색 접근성과 브랜드 설명력이 매우 제한적임 |

### 공통 과제

현재 5개 사이트는 공통적으로 아래와 같은 과제를 가지고 있습니다.

- 검색엔진이 브랜드를 정확히 이해할 수 있도록 돕는 구조화 데이터와 메타 정보가 부족함
- 사이트맵, canonical, Open Graph, 검색 친화 URL 등 기본 SEO 인프라가 불완전함
- FAQ, 강사진, 과정, 정책, 후기 등 검색과 전환에 중요한 정보가 구조화되어 있지 않음
- 일부 사이트는 오래된 CMS 구조 또는 단일 페이지 구조로 인해 확장성과 운영 효율이 낮음
- 멀티 도메인 브랜드 운영에 비해 브랜드 연결성과 엔터티 일관성이 약함

이러한 문제는 단순 운영 이슈라기보다, 사이트 구조 자체를 재정비해야 해결되는 성격에 가깝습니다.

## 3. 왜 게이트 웹사이트만으로는 한계가 있는가

게이트 웹사이트는 5개 브랜드를 한눈에 소개하는 측면에서는 분명 의미가 있습니다.  
다만 검색 성과와 문의 전환을 기준으로 보면, 게이트 사이트만으로는 해결하기 어려운 한계가 있습니다.

### 1. 실제 검색은 브랜드별로 일어납니다

사용자는 "연기학원", "입시학원", "아역배우 교육", "청소년 방송연기", "40대 연기수업"처럼 목적이 분명한 상태로 검색합니다.

이때 검색 결과에서 중요한 것은 게이트 사이트가 아니라, 각 검색 의도에 정확히 대응하는 브랜드별 공식 사이트입니다.

즉,

- 입시를 찾는 고객은 입시센터 페이지에서 합격 정보와 커리큘럼을 확인해야 하고
- 키즈/하이틴 고객은 보호자가 신뢰할 수 있는 정책, 후기, 프로그램 구조를 확인해야 하며
- 애비뉴 고객은 40세 이상 특화 프로그램의 차별성과 상담 동선을 바로 확인해야 합니다

게이트 사이트는 브랜드 소개에는 유용하지만, 이러한 개별 검색 수요를 직접 받아내는 구조는 아닙니다.

### 2. 게이트가 생겨도 기존 사이트의 구조 문제는 남습니다

현재 과제는 "브랜드를 소개할 공간이 부족하다"기보다, "각 사이트가 검색엔진에 최적화된 구조를 갖추지 못했다"는 데 있습니다.

따라서 게이트 사이트를 추가하더라도 아래 문제는 그대로 유지됩니다.

- 기존 사이트의 검색 노출 한계
- 개별 브랜드 페이지의 신뢰 요소 부족
- 문의 전환 페이지의 설득력 부족
- 브랜드별 콘텐츠 확장성 부족

결과적으로 게이트는 브랜드 포트폴리오 소개 기능은 할 수 있지만, 핵심 검색 자산 자체를 회복해주지는 못합니다.

### 3. 게이트는 보조 채널로는 적합하지만, 본 해결책은 아닙니다

게이트 사이트는 향후 기업 소개나 브랜드 허브 관점에서 충분히 활용할 수 있습니다.  
다만 현재 단계에서 우선순위는 게이트 신설보다, 각 브랜드 공식 사이트의 구조를 재정비하는 것에 두는 것이 더 효과적입니다.

## 4. 리뉴얼이 필요한 이유

### 1. 검색 자산을 브랜드별로 축적할 수 있습니다

리뉴얼을 통해 각 사이트에 아래 요소를 체계적으로 구축할 수 있습니다.

- 브랜드별 핵심 키워드 대응 페이지
- FAQ, 과정 소개, 강사진, 후기, 정책 페이지 구조화
- 검색엔진 친화 메타 정보와 구조화 데이터
- 브랜드별 전환 동선과 문의 유도 구조
- 향후 콘텐츠 발행이 가능한 운영 체계

이렇게 되면 단순 노출 개선이 아니라, 브랜드별 검색 자산이 지속적으로 쌓이는 기반이 생깁니다.

### 2. 브랜드 신뢰도와 전환율을 함께 개선할 수 있습니다

현재 검색 환경에서는 단순히 예쁜 웹사이트보다, 아래 정보가 명확한 사이트가 더 높은 신뢰를 얻습니다.

- 어떤 기관인지
- 어떤 대상에게 어떤 서비스를 제공하는지
- 강사진과 운영 주체는 누구인지
- 가격, 정책, 절차, 후기, 성과가 어떻게 정리되어 있는지

리뉴얼은 이러한 요소를 브랜드별로 정리하여 검색엔진과 사용자 모두에게 더 신뢰도 높은 공식 사이트를 제공할 수 있게 합니다.

### 3. 향후 마케팅 효율이 좋아집니다

리뉴얼 이후에는 SEO뿐 아니라 광고, 콘텐츠 마케팅, 후기 운영, CRM 연계 등 후속 마케팅 활동의 효율도 높아집니다.

현재 구조에서는 유입을 늘려도 도착 페이지의 정보 설계와 전환 구조가 약해 효율 개선 폭이 제한될 수 있습니다.  
반면 리뉴얼된 구조에서는 검색 유입, 광고 유입, 재방문 유입 모두를 더 안정적으로 전환으로 연결할 수 있습니다.

### 4. 운영성과 확장성을 함께 확보할 수 있습니다

리뉴얼은 일회성 제작이 아니라, 이후의 운영을 위한 기반 재정비입니다.

- 신규 콘텐츠 추가가 쉬워지고
- 브랜드별 정보 업데이트가 명확해지며
- SEO 개선 작업도 지속적으로 반영할 수 있고
- 향후 게이트 사이트나 캠페인 랜딩도 더 자연스럽게 연결할 수 있습니다

## 5. 제안 방향

### 제안 원칙

이번 프로젝트는 `게이트 사이트 신설`보다 `브랜드 공식 사이트 리뉴얼`을 우선하는 방향으로 제안드립니다.

핵심 이유는 아래와 같습니다.

- 검색 성과는 결국 각 브랜드 사이트에서 발생하기 때문
- 현재 문제는 소개 부족이 아니라 구조 부족이기 때문
- 리뉴얼이 장기적으로 검색, 전환, 운영 측면에서 더 높은 효율을 만들 수 있기 때문

### 제안 옵션 비교

| 구분 | 게이트 웹사이트 | 5개 사이트 리뉴얼 |
| --- | --- | --- |
| 초기 구축 부담 | 낮음 | 상대적으로 높음 |
| 브랜드 소개 기능 | 좋음 | 좋음 |
| 브랜드별 검색 대응 | 제한적 | 높음 |
| 실제 문의 전환 개선 | 제한적 | 높음 |
| 기존 구조 문제 해결 | 어려움 | 가능 |
| 콘텐츠 확장성 | 중간 | 높음 |
| 장기 운영 효율 | 중간 | 높음 |
| 향후 SEO/GEO 대응력 | 제한적 | 높음 |

### 결론

게이트 사이트는 필요 시 이후에 충분히 검토할 수 있는 보조 프로젝트입니다.  
다만 현재 가장 우선해야 할 과제는 5개 브랜드 사이트 자체의 리뉴얼입니다.

## 6. 권장 추진 방식

### 방식 A. 5개 브랜드 동시 리뉴얼

가장 이상적인 방식입니다.  
브랜드 전체의 정보 구조와 디자인 체계를 함께 재정비할 수 있어, 통합된 브랜드 경험과 운영 효율을 확보하기 좋습니다.

적합한 경우:

- 전체 브랜드 체계를 한 번에 정리하고자 할 때
- 중장기적으로 통합 운영 구조를 가져가고자 할 때
- 디자인 시스템과 콘텐츠 체계를 함께 정비하고자 할 때

### 방식 B. 우선순위 기반 단계형 리뉴얼

현실적으로 가장 권장드리는 방식입니다.  
우선순위가 높은 사이트부터 리뉴얼한 뒤, 구조와 성과를 기반으로 나머지 브랜드로 확장하는 방식입니다.

우선 검토 권장 순서는 아래와 같습니다.

1. 하이틴센터
2. 애비뉴센터
3. 입시센터
4. 키즈센터
5. 아트센터

이 순서는 현재 구조적 보완 필요도와 검색 개선 여지를 함께 고려한 기준입니다.

### 방식 C. 리뉴얼 이후 게이트 사이트 추가

브랜드 포트폴리오 소개, 기업 PR, 통합 브랜딩 목적이 필요할 경우에는 리뉴얼 이후 게이트 사이트를 별도 구축하는 방식이 적합합니다.

이 경우 게이트 사이트는 각 브랜드 사이트의 약점을 보완하는 수단이 아니라, 이미 정비된 브랜드 자산을 연결하고 확장하는 역할을 하게 됩니다.

## 7. 리뉴얼 범위 제안

브랜드별 상세 범위는 추후 협의를 통해 조정할 수 있으나, 기본적으로 아래 영역을 포함하는 방향을 권장드립니다.

### 1. 구조 개편

- 브랜드별 정보 구조 재설계
- 검색 친화 URL 및 페이지 체계 정비
- 사용자 흐름에 맞춘 메뉴 및 랜딩 구조 개편

### 2. SEO/GEO 기반 구축

- 메타 정보 체계 정비
- 구조화 데이터 적용
- 사이트맵, canonical, Open Graph, robots 정책 정비
- AI 검색 환경을 고려한 정보 구조 설계

### 3. 콘텐츠 구조화

- FAQ, 강사진, 과정 소개, 후기, 정책 페이지 정비
- 브랜드별 핵심 키워드에 대응하는 설명 구조 마련
- 향후 게시형 콘텐츠 운영이 가능한 기반 설계

### 4. 전환 구조 개선

- 상담 신청 및 문의 흐름 정리
- 브랜드별 CTA 재구성
- 신뢰 요소와 안내 요소 강화

### 5. 운영 확장성 확보

- 관리 가능한 템플릿 구조 설계
- 향후 SEO 운영과 콘텐츠 발행이 가능한 방식으로 구축

- 브랜드별 검색 노출 기반 강화

### 전환 측면

- 문의 전환 페이지 설득력 향상

- 상담 유입 품질 개선

### 운영 측면

- 정보 업데이트와 콘텐츠 운영 효율 개선
- 향후 SEO 운영, 광고 운영, 콘텐츠 마케팅 확장 용이
- 멀티 브랜드 운영 체계 정리

### 브랜드 측면

- 각 브랜드의 독립성과 전문성 강화
- BNB Industry 전체 브랜드 신뢰도 향상
- 향후 통합 브랜드 전략 수행 기반 확보

## 9. 제안드리는 다음 단계

### Step 1. 우선순위 및 범위 확정

브랜드별 중요도와 사업 우선순위를 기준으로 1차 리뉴얼 대상 사이트를 확정합니다.

### Step 2. 상세 진단 및 리뉴얼 전략 수립

선정된 사이트를 기준으로 정보 구조, 콘텐츠 구조, SEO/GEO 개선 항목, 전환 구조를 상세 설계합니다.

### Step 3. 디자인 및 개발 리뉴얼 진행

브랜드 방향성과 검색 구조를 함께 반영한 형태로 리뉴얼을 진행합니다.

### Step 4. 운영 가이드 및 확장 계획 수립

리뉴얼 이후 콘텐츠 운영, SEO 운영, 후속 브랜드 확장 계획까지 함께 정리합니다.

## 10. 맺음말

현재 5개 사이트는 브랜드 경쟁력이나 콘텐츠 자산이 부족해서 성과가 약한 것이 아니라, 그 강점을 충분히 전달하지 못하는 구조적 한계를 가지고 있습니다.

따라서 이번 프로젝트는 단순한 웹사이트 개편이 아니라,  
각 브랜드의 검색 자산, 신뢰 자산, 전환 자산을 다시 세우는 작업으로 접근하는 것이 가장 효과적입니다.

저희는 게이트 사이트보다, 각 브랜드 공식 사이트의 리뉴얼이 보다 장기적이고 실질적인 성과를 만들 수 있는 방향이라고 판단합니다.

필요 시 게이트 사이트는 그 이후 단계에서 브랜드 포트폴리오 허브로 확장 검토하시는 것을 권장드립니다.
