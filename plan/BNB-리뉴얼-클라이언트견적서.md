# BNB Industry 웹사이트 리뉴얼 제안 견적서

> 제안일: 2026년 3월 | 제안사: (에이전시명) | 수신: BNB Industry

---

## 프로젝트 개요

BNB Industry 소속 5개 브랜드 센터의 웹사이트를 새롭게 리뉴얼합니다.
비용 효율성을 위해 **1개 마스터 사이트를 기반으로 5개 브랜드 사이트를 구축**하는 방식으로 제안드립니다.

| 항목 | 내용 |
|------|------|
| 대상 사이트 | 아트센터 / 에비뉴센터 / 입시센터 / 키즈센터 / 하이틴센터 (5개) |
| 리뉴얼 방식 | 마스터 템플릿 기반 멀티브랜드 구축 |
| 디자인 | 각 센터 브랜드 아이덴티티 재정립 포함 |
| 기술 | 최신 웹 프레임워크 + 헤드리스 CMS (통합 콘텐츠 관리) |
| 예상 기간 | 약 6~7개월 |

---

## 왜 마스터 템플릿 방식인가요?

5개 센터 사이트를 분석한 결과, **전체 126개 페이지 중 60% 이상이 동일한 구조**를 공유하고 있습니다.

```
5개 사이트를 각각 개발할 경우    →   약 1억 5천~2억원, 약 10개월 이상
마스터 템플릿 방식               →   약 1억 1천~1억 3천만원, 약 6~7개월
```

**약 40% 비용 절감 + 3개월 이상 기간 단축**이 가능합니다.

마스터 템플릿에서 공통 요소(디자인, 기능, 구조)를 한 번만 개발하고, 각 센터별 고유 콘텐츠와 브랜드 색상만 적용하는 방식입니다.

---

## 핵심 개선 사항

### 1. 브랜드 아이덴티티 재정립
각 센터별로 차별화된 BI(Brand Identity)를 새롭게 수립합니다.
- 로고 / 심볼 / 메인 컬러 / 서브 컬러 재정립
- 톤앤매너 가이드 (사진 스타일, 폰트, 아이콘 등)
- 브랜드 가이드라인 문서 제공

### 2. SEO/AI 검색 최적화 (GEO)
현재 5개 사이트의 SEO 점수는 **20~36점 / 100점** 수준으로 매우 열악합니다.
리뉴얼 후 AI 검색 엔진(ChatGPT, Perplexity 등)에서도 인용되는 구조로 개선합니다.

- 검색 친화적 URL 구조로 전환 (기존 쿼리스트링 방식 → 클린 URL)
- 구조화 데이터(Schema.org) 전 페이지 적용
- 사이트맵, robots.txt, llms.txt 완비
- Core Web Vitals 최적화 (Google 성능 지표)

### 3. 통합 상담 신청 시스템
현재 각 사이트에 분산된 상담 폼을 **통합 관리 시스템**으로 개편합니다.

- 어느 사이트에서 접근해도 5개 센터 중 원하는 곳 선택 가능
- 상담 접수 즉시 담당자에게 알림 발송
- 하나의 통합 관리자 페이지에서 전 센터 상담 현황 조회

### 4. 직접 관리 가능한 CMS
개발사 의존 없이 클라이언트가 직접 콘텐츠를 수정/업데이트할 수 있는 **헤드리스 CMS**를 제공합니다.

- 강사 소개, 공지사항, 갤러리, 합격 현황 등 직접 업데이트
- 5개 센터를 하나의 어드민에서 통합 관리
- 비개발자도 사용 가능한 직관적인 에디터
