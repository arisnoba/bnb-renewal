## 개발 타임라인

```
Phase 1 (8~10주): 마스터 템플릿 + 하이틴센터
├─ 1~2주: 브랜딩 리디자인 방향성 + IA/와이어프레임 확정
├─ 3~4주: 디자인 시스템 + 공통 16종 + 메인 디자인
├─ 5~6주: Next.js + Payload CMS 셋업 + 공통 컴포넌트 개발
├─ 7~8주: 하이틴센터 전체 개발 + CMS 연동 + 콘텐츠 입력
└─ 9~10주: SEO/GEO 셋업 + 통합 상담폼 + QA → 하이틴 런칭

Phase 2 (4~5주): 아트센터 + 키즈센터 병렬
├─ 1주: 브랜드 색상/이미지 적용
├─ 2~3주: 고유 5종씩 디자인 + 개발
└─ 4~5주: 콘텐츠 입력 + SEO + QA → 런칭

Phase 3 (5~6주): 입시센터
├─ 1~2주: 고유 13종 디자인
├─ 3~4주: 개발 + 합격 데이터 구조화
└─ 5~6주: 콘텐츠 + SEO + QA → 런칭

Phase 4 (3~4주): 에비뉴센터
├─ 1주: SSL 정비 + 랜딩 디자인
├─ 2~3주: 개발 + 서브페이지 + SEO
└─ 4주: QA → 런칭

Phase 5 (2주): 통합 QA + 전체 런칭 점검
├─ 301 리다이렉트 전체 검증
├─ 5개 Search Console 등록 + 사이트맵 제출
└─ 클라이언트 CMS 교육 + 운영 가이드 전달
─────────────────────────────────────
총: 22~27주 (5.5~7개월)
```
