## 아트센터

### 입학절차

| 단계 | 제목 | 내용 |
|------|------|------|
| 01 | 상담예약 | 온라인 과정 확인, 온라인 상담신청, 전화 상담신청으로 방문 상담 일정을 예약합니다. |
| 02 | 학원 방문상담 | 예약 일시에 아트센터 1층 안내데스크로 방문해 상담 어플리케이션을 작성한 뒤, 입학상담실에서 시스템, 커리큘럼, 수업 방향을 안내받습니다. |
| 03 | 등록대기 및 반 배정 | 희망 반에 T.O가 있으면 바로 등록할 수 있고, T.O가 없으면 등록대기를 신청합니다. 정규 개강기간에는 정원 내 선착순으로 등록대기가 진행됩니다. |
| 04 | 수강등록 | 입학 당일 수업시간 15~30분 전에 방문해 지문과 사진을 등록하고 수강료를 납부하면 등록이 완료됩니다. 등록 후 카카오톡, 네이버 밴드 초대와 스타카드 발급이 진행됩니다. |
| 05 | 첫 수업 | 담당직원의 안내에 따라 강의실로 이동하고, 담임선생님 소개 후 트레이닝을 시작합니다. |

| 상담 방식 | 기준 |
|-----------|------|
| 온라인 과정 확인 | 희망 Class, 교육시작일, 교육횟수, 교육시간대를 확인한 뒤 수강상담을 신청합니다. |
| 온라인 상담신청 | 방문 희망 일자, 시간, 신청자 정보를 입력합니다. |
| 전화 상담신청 | 대표전화 1577-9929, 내선번호 1번, 신규상담 1번으로 예약합니다. |

## 입시센터

### 입학절차

입시센터는 현재 확인된 레거시 입학절차 원문이 없습니다. 퍼블리싱 기준은 아트, 하이틴, 키즈센터의 공통 입학 흐름을 입시센터 상담 구조에 맞춰 정리합니다.

| 단계 | 제목 | 내용 |
|------|------|------|
| 01 | 상담예약 | 입시반, 예비입시반 등 과정을 확인한 뒤 온라인 상담신청 또는 전화 상담신청으로 상담 일정을 예약합니다. |
| 02 | 학원 방문상담 | 예약 일시에 방문해 상담 어플리케이션을 작성하고, 입시 목표와 현재 준비 상태를 바탕으로 과정, 커리큘럼, 수업 방향을 안내받습니다. |
| 03 | 등록대기 및 반 배정 | 희망 과정과 반에 T.O가 있으면 바로 등록할 수 있고, T.O가 없으면 등록대기를 신청합니다. |
| 04 | 수강등록 | 수업 시작 전 수강료를 납부하면 등록이 완료됩니다. 등록 후 담당자 안내와 스타카드 발급을 진행합니다. |
| 05 | 첫 수업 | 담당직원의 안내에 따라 강의실로 이동하고, 담임선생님 소개 후 입시 트레이닝을 시작합니다. |

| 상담 방식 | 기준 |
|-----------|------|
| 과정 확인 | `/exam#curriculum` 기준으로 입시반, 예비입시반 과정을 확인합니다. |
| 온라인 상담신청 | `/exam/consult` 기준으로 입시 상담을 신청합니다. |
| 전화 상담신청 | 대표전화 1577-9929, 내선번호 2번으로 예약합니다. |

## 퍼블리싱 반영 메모

- 네 센터 모두 화면 섹션은 `입학절차`, `수강료/장학제도`, `휴학/복학/수료`, `환불정책` 순서로 통일합니다.
- 키즈센터는 원문 명칭이 `휴학/복학/보강`이므로 화면 탭에는 공통 명칭을 쓰되, 섹션 내부 제목에서 `보강`을 명확히 드러냅니다.
- 입시센터는 입학절차와 휴학/복학/수료 원문이 없어, 문서에 명시한 임시 구성과 확인 필요 항목을 우선 기준으로 사용합니다.
- 환불정책은 센터별 원문 내용이 사실상 동일하므로 공통 컴포넌트 또는 공통 데이터로 관리할 수 있습니다.
