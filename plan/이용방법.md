## 목적

`/art/how-to-use`의 15개 이용 안내 카드는 현재 모두 `자세히 보기` UI를 동일하게 노출한다. 실제 콘텐츠 성격은 단순 안내, FAQ DB 상세, 기존 상세 페이지 연결이 섞여 있으므로 항목별 행동을 분리한다.

목표는 다음과 같다.

| 구분             | 처리 방향                                                                                |
| ---------------- | ---------------------------------------------------------------------------------------- |
| 링크 제거        | 카드 본문만으로 충분한 항목은 `자세히 보기`를 제거하고 안내 카드로 유지                  |
| FAQ 상세         | 운영 방법, 대상, 시간, 장소, 주의사항이 있는 항목은 FAQ DB의 `이용방법` 카테고리로 이동 |
| 상세 페이지 링크 | 이미 별도 상세 페이지가 있거나 목록/DB가 필요한 항목은 CTA 링크로 연결                   |

## 권장 분류

### 1. 링크 제거 후 카드 안내만 유지

아래 항목은 별도 페이지로 이동할 이유가 약하고, 카드 본문 또는 짧은 보조문만으로 충분하다. 카드에서 `자세히 보기`를 제거하거나 비활성 상태로 두는 것을 권장한다.

| 항목                               | 권장 본문 방향                                                                            | 비고                                                                      |
| ---------------------------------- | ----------------------------------------------------------------------------------------- | ------------------------------------------------------------------------- |
| 배우앤배움 촬영 스튜디오           | 최고급 장비와 시설을 갖춘 BNB 전용 촬영 공간, 관계자 외 출입 제한 안내                    | 상세 제작 절차는 프로필 제작 페이지와 겹치므로 카드에서는 시설 소개만     |
| 드라마 편성표, 시놉시스 비치서비스 | 아트센터 1층 DID에서 드라마 편성 예정 정보와 시놉시스 확인 가능                           | 연도 표기는 삭제하고 현재 제공 서비스처럼 보이도록 문구 정리              |
| 영화 라인업, 시놉시스 비치서비스   | 아트센터 1층 DID에서 영화 라인업과 시놉시스 확인 가능                                     | 현재 코드 설명의 `드라마의 시놉시스` 문구는 `영화의 시놉시스`로 수정 권장 |
| 광고 에이전시 리스트정보 서비스    | 1층 DID에서 주요 광고모델 에이전시 정보를 확인하고, 추가 문의는 EnM 매니지먼트팀으로 연결 | 오프라인 제공 서비스이므로 링크 불필요                                    |

| 학생 프린트 서비스                 | 1층 안내데스크 로비 PC에서 출력 후 프런트 수령, 재학생 무료                               | 현재 코드 설명이 `인쇄.`로 끝나므로 문장 보완 필요                        |

### 2. FAQ DB `이용방법` 카테고리로 이동

아래 항목은 사용자가 실제 이용 전에 확인할 세부 조건이 있다. 카드 CTA는 `/art/faq?category=etc`로 연결하고, FAQ DB에는 `category: "etc"` 값을 유지하되 화면과 관리자 라벨은 `이용방법`으로 표시한다. FAQ 문서는 `centers: ["all"]`로 등록해 모든 센터 FAQ에서 접근 가능하게 한다.

| 항목                                     | FAQ 질문 예시                                | FAQ에 넣을 상세                                                                                                    |
| ---------------------------------------- | -------------------------------------------- | ------------------------------------------------------------------------------------------------------------------ |
| 전용 피트니스 무료제공                   | 전용 피트니스는 어떻게 이용하나요?           | 스타카드 소지자 이용 가능, 휴메이크 휘트니스 논현점 방문 시 스타카드와 본인 확인, 평일/주말 운영시간, 주소, 연락처 |
| 24시간 연습실 이용, 지문 등록 보안서비스 | 운영시간 이후에도 연습실을 이용할 수 있나요? | 평일 09:00-22:00, 주말 10:00-18:00 이후 지문 출입 가능, CCTV 운영, 신규 등록 시 지문 등록, 퇴교 시 자동 삭제       |
| 개인연습실, 그룹스터디룸 제공            | 개인연습실과 그룹스터디룸은 예약해야 하나요? | 1-2인실, 2-3인실 개인연습실과 최대 10명 그룹스터디룸, 반별 과제/회의 활용, 별도 예약 없이 재학생 자유 이용         |
| 배배대본 배포 서비스                     | 배배대본은 언제 어디서 받을 수 있나요?       | 매주 토요일 신규 오디션용 대본 준비, 독백/2인극 구성, 주중 1층 안내데스크에서 요청 후 수령                         |
| 씨네21 잡지 비치                         | 씨네21 잡지는 누가 어디서 이용할 수 있나요?  | 재학생과 Staff 대상, 1층 안내데스크 및 2층 서재, 과월호 확인 가능, 원내에서 이용 가능                              |

### 3. 상세 페이지 링크 유지 또는 연결

아래 항목은 이미 별도 페이지가 있거나, 원문에서 특정 상세 페이지를 안내한다. 카드 CTA를 유지하고 목적지가 명확한 링크로 연결하는 것이 적합하다.

| 항목                               | 권장 CTA              | 연결 후보                 | 이유                                                                               |
| ---------------------------------- | --------------------- | ------------------------- | ---------------------------------------------------------------------------------- |
| 스타카드 발급                      | 스타카드 혜택 보기    | `/art/starcard`           | 현재 별도 스타카드 페이지가 있고 제휴업체 목록/상세 정보가 필요                    |
| 프로필 제작 및 PPT 디자인 서비스   | 프로필 제작 절차 보기 | `/art/profile-production` | 현재 별도 프로필 제작 페이지가 있고 신청/촬영/보정 흐름이 상세함                   |
| 드라마, 영화, 광고 오디션 정보안내 | 캐스팅 현황 보기      | `/art/casting-status`     | 오디션 정보 자체는 오프라인 제공이고, 온라인 연결은 공개 캐스팅 현황 페이지가 적합 |

## 항목별 상세 정리

### 스타카드 발급

- 카드 성격: 상세 페이지 링크형
- 핵심 안내: 신규 등록 시 무료 발급, 분실 재발급 시 비용 발생 가능, 매월 갱신 필요
- FAQ 포함 여부: 카드에는 요약만 두고, 발급/갱신 절차는 `/art/starcard` 또는 FAQ 중 하나로 통일
- 추천 링크: `/art/starcard`

### 전용 피트니스 무료제공

- 카드 성격: FAQ 상세형
- 핵심 안내: 배우앤배움 재학생이 스타카드 확인 후 제휴 피트니스 이용
- FAQ 상세:
  - 이용 대상: 스타카드 소지자
  - 이용 방법: 휴메이크 휘트니스 논현점 프런트에서 스타카드와 본인 확인
  - 이용 가능 시간: 평일 06:00-24:00, 주말/공휴일 08:00-20:00, 둘째/넷째 일요일 휴무
  - 위치: 서울시 강남구 강남대로 546 지하 2층, 논현역 2번 출구 인근
  - 연락처: 02-548-7079
- 추천 링크: 없음. FAQ 앵커만 연결

### 배우앤배움 촬영 스튜디오

- 카드 성격: 링크 제거 안내형
- 핵심 안내: 영상과 사진 촬영이 가능한 BNB 전용 스튜디오, 촬영/제작 전문 인력이 콘텐츠 회의부터 완성까지 지원
- 주의 안내: 대본 리딩 및 캐스팅 관련 촬영 집중을 위해 관계자 외 출입 제한
- 추천 링크: 없음. 프로필 제작 CTA와 중복되면 이 카드는 시설 소개로만 유지

### 프로필 제작 및 PPT 디자인 서비스

- 카드 성격: 상세 페이지 링크형
- 핵심 안내: 매월 제한 인원으로 프로필 제작, 촬영 기획부터 보정/PPT 디자인까지 관리
- 레거시 절차: 1층 안내데스크 신청, 매니지먼트팀 콘셉트 회의, 촬영 당일 매니저 현장 지원, 10장 보정, PPT 제작 후 이메일 수령
- 비용 문구: 비용 상세는 별도 FAQ에서 처리하므로 이 페이지 카드에는 노출하지 않음
- 추천 링크: `/art/profile-production`

### 드라마, 영화, 광고 오디션 정보안내

- 카드 성격: 상세 페이지 링크형
- 핵심 안내: 국내 드라마/영화/광고 오디션 정보를 취합해 보관하고 오프라인으로 안내
- 상세 연결: 오디션 정보 자체는 오프라인 제공 서비스로 유지하고, 온라인 CTA는 공개 캐스팅 현황으로 연결
- 추천 링크: `/art/casting-status`

### 드라마 편성표, 시놉시스 비치서비스

- 카드 성격: 링크 제거 안내형
- 핵심 안내: 1층 DID에서 드라마 편성 예정 정보와 진행 중인 드라마 시놉시스 확인 가능
- 주의 안내: 자료 무단 외부 유출 금지
- 문구 기준: 연도 표기는 삭제하고 현재 상시 제공되는 정보처럼 자연스럽게 정리
- 추천 링크: 없음

### 영화 라인업, 시놉시스 비치서비스

- 카드 성격: 링크 제거 안내형
- 핵심 안내: 1층 DID에서 영화 라인업과 진행 중인 영화 시놉시스 확인 가능
- 주의 안내: 자료 무단 외부 유출 금지
- 문구 기준: 연도 표기는 삭제하고 현재 상시 제공되는 정보처럼 자연스럽게 정리
- 추천 링크: 없음

### 광고 에이전시 리스트정보 서비스

- 카드 성격: 링크 제거 안내형
- 핵심 안내: 1층 DID에서 국내 주요 광고모델 에이전시 50여 곳 정보 확인 가능
- 보조 안내: 그 외 에이전시 정보는 배우앤배움 EnM 매니지먼트팀 문의
- 제공 방식: 오프라인 제공 서비스
- 추천 링크: 없음

### 24시간 연습실 이용, 지문 등록 보안서비스

- 카드 성격: FAQ 상세형
- 핵심 안내: 운영시간 이후에도 지문 인식으로 출입 가능
- FAQ 상세:
  - 기본 운영시간: 평일 09:00-22:00, 주말 10:00-18:00
  - 보안: 사각지대 없는 CCTV와 지문 등록 기반 출입
  - 등록 시점: 최초 신규 등록 과정
  - 휴학 시: 지문 등록 유지
  - 퇴교 시: 자동 삭제
- 추천 링크: 없음. FAQ 앵커만 연결

### 개인연습실, 그룹스터디룸 제공

- 카드 성격: FAQ 상세형
- 핵심 안내: 방음 개인연습실과 그룹스터디룸을 재학생이 자유롭게 이용
- FAQ 상세:

## 구현 메모

### 데이터 구조 제안

현재 `services` 배열에 모든 항목이 동일한 필드만 갖고 있다. 액션 분리를 위해 아래 정도의 필드만 추가하면 충분하다.

| 필드         | 용도                          |
| ------------ | ----------------------------- |
| `actionType` | `none`, `faq`, `link` 중 하나 |
| `href`       | FAQ 카테고리 또는 실제 상세 페이지 링크 |
| `href`       | 실제 상세 페이지 링크         |
| `ctaLabel`   | 링크형 카드의 버튼 문구       |

### 화면 구성 제안

1. 상단 카드 그리드는 유지한다.
2. `actionType: none` 항목은 `자세히 보기`를 표시하지 않는다.
3. FAQ 상세가 필요한 항목은 `/art/faq?category=etc`로 이동시킨다.
4. `actionType: link` 항목은 명확한 CTA 문구와 함께 기존 상세 페이지로 이동한다.
5. 카드 하단 CTA 문구는 모두 `자세히 보기`로 통일하지 말고 행동에 맞춘다.

| 액션 | 버튼 문구 예시                                              |
| ---- | ----------------------------------------------------------- |
| FAQ  | 이용 방법 보기, 운영 기준 보기                              |
| 링크 | 스타카드 혜택 보기, 프로필 제작 절차 보기, 캐스팅 현황 보기 |
| 없음 | 버튼 없음                                                   |

### 확정된 반영 기준

| 항목                 | 반영 기준                                                                  |
| -------------------- | -------------------------------------------------------------------------- |
| 드라마/영화 편성표   | 연도 표기는 삭제하고 현재 제공 서비스처럼 문구를 워싱                      |
| 씨네21 대여 시간     | 요일/시간 정보는 노출하지 않음                                             |
| 프로필 제작 비용     | 새 FAQ에서 처리하고, 이 페이지 링크는 `/art/profile-production`으로 연결   |
| 오디션 정보안내 링크 | 오프라인 제공 서비스로 설명하고, 온라인 CTA는 `/art/casting-status`로 연결 |
| 광고 에이전시 리스트 | 오프라인 제공 서비스로 설명하고 링크는 제공하지 않음                       |
