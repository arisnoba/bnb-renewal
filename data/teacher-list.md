# 통합 티쳐 리스트

> 기준 파일: `artcenter/teacher.md`, `highteen/teacher.md`, `kidscenter/teacher.md`, `exam/teacher.md`  
> 목적: 최종 공개 대상 티쳐와 단일 전역 노출 순서를 한 곳에서 관리한다.

## 정렬 원칙

- 이 파일에 있는 85명만 기본 공개 대상이다.
- 원본 4개 파일에 없는 강사는 `draft`로 둔다.
- 한 강사가 여러 센터에 있으면 row는 1개만 만든다.
- 센터 필터 적용 시에도 정렬은 하나의 `order` 값을 사용한다.
- 필터링은 해당 센터에 없는 강사를 제외할 뿐, 남은 강사의 상대 순서는 유지한다.
- 비입시 센터에 있는 강사는 센터별 원본 순서의 평균값을 기준으로 전역 순서를 만든다.
- 입시센터에만 있는 강사는 별도 성격을 고려해 `501`번부터 배치한다.
- 입시와 다른 센터에 동시에 있는 강사는 중복 row를 만들지 않고 비입시 전역 순서를 따른다.
