# 배우앤배움 선별 이관 세트

이 폴더는 `data/baewoo-split/tables`에서 실제 프로젝트에 필요한 테이블만 우선순위별로 다시 모아둔 작업본이다.

## 생성 방식

- 생성 스크립트: `scripts/curate_baewoo_tables.py`
- 입력: `data/baewoo-split/tables`
- 출력: `p0/`, `p1/`, `p2/`, `reference/`, `summary.json`

## 현재 범위

- 이번 리뉴얼 1차에서 실제 검토/이관 대상으로 보는 데이터는 `p0`, `p1`이다.
- `p2`는 상담/리드 성격의 운영 데이터라 현재 범위에서 제외한다.
- 세부 테이블 구성과 최신 판단은 각 폴더의 `README.md`에 기록한다.

## 폴더 역할

- `p0/`: 사이트 오픈에 필요한 핵심 공개 콘텐츠
- `p1/`: 공개 사이트 확장용 콘텐츠
- `p2/`: 현재 미사용인 운영/개인정보 데이터
- `reference/`: 구조 참고용 자료
