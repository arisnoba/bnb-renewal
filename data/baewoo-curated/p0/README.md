# P0 선별본

이 폴더는 가장 먼저 테스트 이관할 공개 콘텐츠 테이블을 모아둔 곳이다.

## 포함 테이블

- `g5_content`
- `g5_content2`
- `g5_teacher`
- `g5_teacher2`
- `g5_write_new_notice`

## 사용 목적

- 정적 페이지, 강사진, 공지사항처럼 사이트 핵심 공개 데이터를 먼저 옮긴다.
- Postgres 스키마와 변환 규칙을 가장 작은 범위에서 검증한다.

## 현재 판단

- `pages` 계열 데이터는 현재 운영 기준에서 `개인정보처리방침`, `이용약관`만 사용 대상으로 본다.
- 그 외 일반 소개/정적 페이지 데이터는 현재 리뉴얼 범위에서 우선순위가 낮다.
- `g5_teacher`, `g5_teacher2`는 현재 `teachers` 컬렉션으로 정리한다.
- `teachers`, `news` 계열 데이터는 계속 검토 대상으로 유지한다.
