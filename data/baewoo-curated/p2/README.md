# P2 선별본

이 폴더는 운영성 데이터와 개인정보가 포함된 테이블을 모아둔 곳이다.

## 포함 테이블

- `g5_write_new_counsel`
- `g5_write_new_counsel_2`
- `sm_customer`

## 사용 목적

- 상담 이력과 고객 데이터를 별도 검토 후 순차 이관한다.
- 개인정보 보존 범위, 마스킹, 새 서비스 스키마가 먼저 확정되어야 한다.

## 현재 결정

- 이번 리뉴얼 1차 범위에서는 P2를 사용하지 않는다.
- `g5_write_new_counsel`, `g5_write_new_counsel_2`, `sm_customer`는 현재 운영 기준에서 직접 이관하지 않는다.
- 추후 상담/CRM 기능을 별도로 설계할 때만 재검토한다.
