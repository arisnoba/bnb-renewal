# 테이블별 SQL 파일 안내

이 폴더에는 원본 덤프에서 추출한 테이블별 SQL 파일이 들어 있다.

## 포함 내용

- 테이블 생성문
- 해당 테이블의 INSERT 데이터
- 원본에 있던 ALTER 구문

## 사용 목적

- 특정 테이블만 따로 열어 구조와 데이터를 확인한다.
- 이관 후보 테이블을 선별한다.
- 후속 변환 스크립트의 입력 단위를 작게 유지한다.

## 주의

- 여기 있는 파일은 여전히 MySQL 또는 MariaDB 문법 기준이다.
- Postgres에 직접 적재하는 파일이 아니라 검토와 선별용 중간 산출물이다.
